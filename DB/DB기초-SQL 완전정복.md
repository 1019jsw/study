# DB기초-SQL 완전정복

## 1 . DBMS
### RDBMS
1. Relationed DBMS
2. 연관된 데이터들의 집합체 -> relation들의 모임

### RDBMS의 특징
1. SQL(Structured Query Language)지원
2. relation(관계)으로 데이터를 정의함
3. 어떠한 물리적 포인터 없이 복수 테이블에 접근 가능함
4. 보안, 권한, 트랜잭션 관리와 같은 기능을 지원

### RDBMS 용어
* relation
	1. 같은 성격을 가진 데이터들의 집합
	2. 테이블과 같은 의미

* tuple
	1. 속성의 값으로 이루어진 값의 집합
	2. record, row와 같은 의미
